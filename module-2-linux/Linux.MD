Q1. Why Linux is used in DevOps
Answer: Linux is free Open Source OS
Used for running servers,clouds VM, Docker containers, Kubernetes nodes, pipelines, databases

        Benifits
        1. Light, fast and scalable
        2. Easy to automate
        3. Almost all cloud systems runs on Linux
        4. Docker and Kubernetes uses linux kernel features
        5. All DevOps toolchains are built for Linux --> Jenkins, GitLab CI runners, Kubernetes master/worket nodes etc

Q2. Linux folder structure
Answer:
/home --> User home folders
/root --> Root user's home
/etc --> configuration files
/var --> Logs, caches
/usr --> User binaries, libraries
/bin --> essentials commands
/sbin --> system binaries
/opt --> Optional applications
/tmp --> temporary files

Q/A Where do logs live --> /var/logs
Q/A Where are services configured --> /etc
Q/A Where is cron --> /etc/crontab
